# 项目概述 python735

本项目是一个基于Python Django框架的图书管理系统。

## 技术栈

- 后端：Python Django
- 数据库：MySQL

## 功能模块

- 用户管理：登录、注册、登出、修改密码
- 数据操作：添加图书、作者、出版社
- 数据查询：
  - 查找出版社信息
  - 查看图书信息
- 安全性：
  - 密码加密
  - 信息唯一性判断

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
